# Literature Map

## Visual Servoing

Image-based and position-based visual servoing provide strong local correction but can fail under occlusion, poor conditioning, and geometry mismatch.

## Foundation Policies

Foundation policies provide semantic action proposals but can produce locally unsafe actions when visual error or geometry uncertainty is high.

## Hybrid Visual Control

Hybrid controllers and shields are the closest prior family. Paper 116 narrows the contribution to calibrated arbitration between foundation decoding and servo correction using action-critical visual error and stability margins.
