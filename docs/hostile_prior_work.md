# Hostile Prior Work

The hostile set contains 100 papers from the shared robotics literature pool. The strongest threats are visual servoing, hybrid visual control, sliding-mode visual control, and foundation-policy safety:

- Constrained Image-Based Visual Servoing of Robot Manipulator with Third-Order Sliding-Mode Observer (2022)
- Hybrid Deep Learning Framework for Eye-in-Hand Visual Control Systems (2025)
- A Hybrid Visual Servo Controller for Robust Grasping by Wheeled Mobile Robots (2009)
- Picking Robot Visual Servo Control Based on Modified Fuzzy Neural Network Sliding Mode Algorithms (2019)
- Hybrid Quantum-Classical Algorithms for High-Precision Sensor Fusion in Robotics Applications (2025)

## Novelty Pressure

Visual servoing and hybrid control are old and crowded. The defensible contribution is narrower: calibrated arbitration between learned foundation-policy decoding and classical servo correction under visual/geometry shift.

## Evidence Boundary

The local benchmark supports this boundary against foundation-only, IBVS-only, PBVS, fixed hybrid switching, uncertainty gating, CBF shielding, and ensemble risk gating. It does not settle real hardware generality.
